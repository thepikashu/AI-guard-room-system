# AI Guard Room Agent

An AI-based security system that combines face recognition, voice authentication, and intelligent dialogue to control access to restricted spaces.

## Key Features

- Face Recognition (DeepFace, VGG-Face)
- Passcode Authentication via Speech
- 3-Level Escalating Dialogue System
- Force Entry Detection
- Real-time Video Processing

## System Pipeline 

Video Input → Face Detection → Face Recognition  → Authorized → Access Granted  → Unauthorized → Escalation Dialogue → Passcode / Security Alert
